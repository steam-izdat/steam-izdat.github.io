# 🇷🇸 Сербия

<div class="cl cl-note" markdown>

**Кому подходит.** Только при реальном переезде. У Сербии нет договора с США → налог с продаж в США 30% (как у РФ), налоговой выгоды нет. USD-SWIFT из сербских банков проводит в основном Райффайзен.

</div>

## 🧾 Налоги

**Налог США (Withholding Tax): 30%.** <span class="stamp stamp--date">на Июнь 2026</span> У Сербии нет соглашения с США об избежании двойного налогообложения, поэтому с доли продаж в США взимается дополнительный налог 30% (не путай с комиссией Steam). Снизить его через W-8BEN не получится.

**Местные налоги.** В Сербии действуют стандартные европейские налоги. Льготы применяются только к доходу до 4000 евро в месяц с лимитом до ~48000 евро в год. При превышении этой суммы ставка налога составляет более 40%.

## 🏦 Банки и приём выплат Steam

У многих банков Сербии есть проблемы с приёмом SWIFT-платежей в долларах, за исключением Райффайзен-банка.

<div class="cl cl-warn" markdown>

Проверенных кейсов приёма именно выплат Steam по сербским банкам в сообществе мало. Появился свежий живой кейс — поделись в [чате STEAMиздат](https://t.me/+qWcZc4g1arRhMGY6).

</div>

## ⚠️ Риски и подводные камни

Рассматривать Сербию стоит только при планируемом постоянном переезде в страну — как канал для приёма выплат без переезда смысла нет (налог 30%, банки капризные к USD-SWIFT). Если переезжаешь, ознакомься с гайдом: [srb.guide](https://www.srb.guide/)
