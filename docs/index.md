<div class="cl cl-danger" markdown>

**Важное обновление · <span class="stamp stamp--date">актуально на 2026-06</span>**

**Челябинвестбанк (ЧИБ) под санкциями с мая 2026** — попал в 20-й пакет санкций ЕС, SWIFT-выплаты от Valve через него больше не проходят. В РФ остался **только Райффайзенбанк**, и он работает **исключительно с ИП** — физлицам и самозанятым счёт для Steam-выплат сейчас не открывают.

Если ты физлицо или самозанятый — рабочего РФ-банка для приёма выплат от Steam на данный момент **нет**: вариант либо перейти в статус ИП ради Райфа, либо работать через зарубежную юрисдикцию. Подробно: [Работа из РФ](02-russia/index.md).
